## Introduction

This is a portfolio website and blog for my portfolio. It is built with Next.js and uses the Strapi CMS to manage the content.

The Next.JS application and the Strapi CMS are deployed on my own VPS with CONTABO.

## Technical Architecture Overview

#### Frontend Stack

Next.js 14 Implementation

- Using the latest Next.js 14 with App Router architecture
- Implemented static site generation (SSG) for optimal performance
- Leveraging Next.js's built-in image optimization with next/image
- Utilizing route handlers for API endpoints
- Implementing proper metadata for SEO optimization

#### UI/UX Implementation

- Modern component architecture using React 18
- Tailwind CSS for styling with custom configuration
- Shadcn UI components for consistent design system
- Framer Motion for smooth animations
- Responsive design with mobile-first approach
- Dark mode support using next-themes

#### Performance Optimizations

- Image optimization with Next.js Image component
- Font optimization using next/font
- Code splitting and lazy loading
- Static page generation where possible
- Client-side navigation for smooth transitions

### Backend Stack

#### CMS Architecture

- Strapi v5 as headless CMS
- RESTful API integration
- Proper error handling and type safety
- Efficient data fetching with query optimization

#### Database

- PostgreSQL as database, also hosted on VPS

### Data Fetching Patterns

```js
// Example of optimized Strapi query
const url =
  `${CMS_URL}/api/posts?` +
  qs.stringify(
    {
      fields: ['slug', 'title', 'subtitle', 'publishedAt'],
      populate: {
        categories: {
          fields: ['title', 'slug']
        },
        image: {
          fields: ['url']
        }
      },
      sort: ['publishedAt:desc'],
      pagination: { pageSize, page }
    },
    { encodeValuesOnly: true }
  );
```
